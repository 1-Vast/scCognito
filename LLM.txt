============================================================
scCognito: PLM–LLM 协同生物认知框架 (v2)
Biological Cognitive Intelligence Framework
============================================================

------------------------------------------------------------
0. 核心设计原则 (Design Principle)
------------------------------------------------------------

系统优化遵循两个基本原则：

(1) 职责解耦 (Role Decoupling)
    Teacher 负责语义监督生成
    PLM 负责结构表示学习
    Agent 负责策略调整与闭环优化

(2) 正确读取表示 (Representation First)
    生物 foundation model 的性能瓶颈
    不在模型规模，而在表示汇总与对齐方式。

系统目标：
    learn stable biological manifold
    rather than memorizing datasets.


------------------------------------------------------------
1. 基本认知与设计理念
------------------------------------------------------------

生物智能建模是两个互补空间的协同学习：

(1) PLM 学习 (Structural World Model)
    连续 biological manifold
    - 空间拓扑
    - 表达流形
    - 多尺度组织结构
    - 跨数据泛化

(2) LLM 学习 (Semantic Reasoning Engine)
    离散 biological semantic space
    - pathway / marker / mechanism knowledge
    - 推理与解释
    - 科学决策生成

形成认知闭环：

    Perception (PLM)
        ↕
    Alignment (Bridge / SER)
        ↕
    Cognition (LLM Agent)


------------------------------------------------------------
2. 数据范式 (Dataset Paradigm)
------------------------------------------------------------

训练数据分为两类：

(1) Foundation Corpus
    多来源空间数据用于预训练：
        - 多物种
        - 多组织
        - 多技术平台

目标：
    学习 biological manifold，而非特定数据集。

(2) Target Evaluation Domain
    用于微调与指标评估的数据。
    评估仅在目标域内部进行，
    不跨数据集计算指标。


------------------------------------------------------------
3. PLM: Biological Representation Model
------------------------------------------------------------

PLM 构建 biological foundation embedding。

3.1 Biological Tokenization
统一编码：
    expression + spatial + context signals

3.2 Dual Stream Graph Learning
    G_spatial   : 物理空间拓扑
    G_attribute : 表达流形邻接

    Z = f_theta(X, G_spatial, G_attribute)

3.3 Self-supervised Objectives
    - masked reconstruction
    - neighbor prediction
    - relation consistency
    - augmentation invariance

3.4 Multi-scale Representation (NEW)
生物结构天然多尺度：

    Z_local   = GNN(k_small)
    Z_mid     = GNN(k_medium)
    Z_global  = GNN(k_large)

    Z = concat(Z_local, Z_mid, Z_global)

避免 layer fragmentation。


------------------------------------------------------------
4. 表示读取原则 (Representation Aggregation Principle)
------------------------------------------------------------

生物信号为 distributed signal，
不存在单一 semantic token。

默认 pooling：

    Z_global = mean(Z_tokens)

推荐扩展：

    w = softmax(score(Z_tokens))
    Z_global = Σ w_i * Z_i

禁止默认 CLS pooling。

Embedding normalization 必须保持跨切片稳定。


------------------------------------------------------------
5. SER: 语义-结构协同学习 (Semantic-Structural Co-learning)
------------------------------------------------------------

Semantic Energy Regularization:

    L_total = L_structure + λ * (E_pos + w_neg * E_neg)

对象：
    C : semantic confidence (由Teacher提供)
    P : semantic prototypes (可学习的原型向量)
    Z : PLM embedding (细胞表征)

SER 核心机制：
    (1) 正向对齐 (Positive Alignment, E_pos):
        拉近 Z 与目标语义原型靶点 (C @ P) 的余弦相似度。
    (2) 负向排斥 (Negative Repulsion, E_neg) [NEW]:
        推开 Z 与无关语义原型的距离，利用负采样对抗表示坍缩 (Representation Collapse)。

SER 作用：
    - 重塑 manifold 几何
    - 调控拓扑传播
    - 限制搜索空间至 biologically plausible 区域
    - 维持细胞异质性，防止模式坍缩 (Mode Collapse)


------------------------------------------------------------
6. Teacher / Student / Agent 职责闭环
------------------------------------------------------------

(1) LLM Teacher
    ONLY 生成监督信号：
        - prototypes
        - pseudo labels
        - semantic constraints
    不参与数据读取。

(2) PLM Student
    学习稳定结构 embedding。

(3) LLM Agent
    仅允许可审计动作：
        - 超参调整
        - 训练策略
        - preprocessing
        - pooling 选择

禁止黑盒调参。


------------------------------------------------------------
7. Spatial Inductive Bias
------------------------------------------------------------

Foundation models 不会自动学习高阶组织结构。

必须显式加入：

    spatial_energy_loss
    topology_consistency
    layer_smooth_constraint

Example:
    loss += λ * spatial_smoothness(Z, graph)


------------------------------------------------------------
8. 训练动力学与防坍缩原则 (Training Dynamics & Anti-Collapse) [NEW]
------------------------------------------------------------

生物流形极易在强语义约束下发生坍缩。必须遵循以下原则：

(1) 语义退火 (Semantic Annealing):
    遵循先结构、后语义的认知规律。早期 (如 0-20% epoch) 必须让模型纯依靠 L_structure 学习原始拓扑。
    中后期通过平滑曲线 (如二次方 t^2) 逐渐引入语义权重 λ。绝对禁止初始阶段给予强语义约束。

(2) 结构保真度 (Structural Fidelity):
    自监督重构损失 (Reconstruction Loss) 是维持流形底座的核心正则化器。
    若观测到坍缩，应降低 mask_ratio 暴露更多底层特征，并降低 spatial_k 以防止过度平滑。

(3) 空间张力 (Spatial Tension):
    必须利用 SER 中的负向排斥 (Negative Repulsion) 撑开特征空间，确保嵌入表征的多样性。


------------------------------------------------------------
9. Context Scaling Principle
------------------------------------------------------------

更大邻域 ≠ 更好表示。

推荐：
    multi-scale context

避免：
    单纯扩大 radius 或 kNN。


------------------------------------------------------------
10. Agent Evaluation Protocol
------------------------------------------------------------

Agent 决策基于统一指标：

(1) Stability
    NaN / gradient norm / convergence

(2) Representation Quality
    silhouette / CH / DB / fragmentation ratio

(3) Semantic Consistency
    SER energy / prototype coverage

(4) Light Downstream Validation
    target-domain clustering metrics
